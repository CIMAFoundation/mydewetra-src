# Source Code of MyDewetra Platform
This repository contains the source code of the MyDewetra platform

**portal-backend.zip** contains the backend of the platform, written in Python.

**dewetra3-backend.zip** contains the backend of the Dewetra Application, written in Python.

**ngx-cima-dewetra3-src.zip** contains the frontend of the Dewetra Application, written in Angular.
